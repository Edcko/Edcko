## 🧑‍💻 Sobre mí

Me apasiona construir software que resuelve problemas reales. No me motivan los proyectos escolares ni los tutoriales: me meto de lleno en sistemas que la gente usa todos los días.

En los últimos años he desarrollado desde cero plataformas completas —desarrollo frontend, backend, base de datos y despliegue— para gimnasios, spas y e-commerce. He integrado Stripe, Mercado Pago, he configurado servidores con Docker y Nginx, y he trabajado con arquitecturas multi-tenant donde cada cliente tiene sus datos aislados.

Me gusta entender cómo funciona todo, desde la base de datos hasta el CI/CD. No soy de copiar y pegar sin saber por qué.

## 🚀 Proyectos en Producción

### 🏋️ Sistema de Gestión para Gimnasios - BNKR
**[Ver en producción →](https://www.bunkerboxgym.com)**

Desarrollé desde cero un sistema completo para administrar gimnasios con múltiples sucursales. El proyecto incluye frontend, backend, base de datos y toda la infraestructura de despliegue.

**Lo que construí:**
- 🏢 **Multi-sucursal** - Cada gym tiene sus datos aislados, pero se administran desde un solo lugar
- 💳 **Pagos con Stripe** - Integración completa para membresías y renovaciones
- 📅 **Reservas y horarios** - Sistema de clases con control de capacidad e instructores
- 👥 **6 niveles de acceso** - Desde Super Admin hasta Cliente, con permisos específicos
- 📊 **Dashboard gerencial** - Estadísticas, reportes y métricas en tiempo real
- 📦 **Inventario inteligente** - Alertas automáticas cuando el stock baja

**Stack:** Vue.js 3 · Node.js · TypeScript · PostgreSQL · Prisma · Stripe · Socket.io · Docker · Nginx

### 💆 Sistema de Gestión para Spas de Belleza - Cherry
**[Ver en producción →](https://www.gpocherry.com/login)**

Plataforma enterprise para cadenas de spas. Este proyecto me retó a diseñar una arquitectura multi-tenant robusta con calendarios inteligentes y sincronización en tiempo real entre sucursales.

**Lo que construí:**
- 📅 **Calendario inteligente** - Bloqueo automático de cabinas, validación de conflictos, filtros por terapeuta
- 👥 **CRM completo** - Historial de cada cliente: sesiones, compras, pagos a plazos (abonos)
- 🏢 **Multi-tenant** - Arquitectura que escala a N sucursales sin duplicar código
- 🔐 **8 niveles de roles** - Desde Desarrollador hasta Terapeuta, con permisos granulares
- 📊 **Reportes exportables** - Generación de PDFs y Excel para contabilidad y gerencia
- ⚡ **Tiempo real** - Socket.io para que los cambios se vean instantáneamente en todas las pantallas

**Stack:** Vue 3 · Pinia · Vuetify · Node.js · Express · PostgreSQL · Socket.io · JWT
